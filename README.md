# Open FileTrucker

## **＊詳しい情報は、[公式サイト](https://corenion.github.io/file_trucker/)をご覧ください。**

# Features

- シンプルなUI
- クロスプラットフォーム
- ローカルネットワークを利用
- オープンソース
